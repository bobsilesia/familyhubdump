# Services

## familyhub.upload_media
Uploads a file to the device via SmartThings. Parameters: `path`, `content_type`.

## familyhub.execute
Sends arbitrary commands (component, capability, command, arguments).

## familyhub.set_ice_maker
Sets ice maker state (`state`: true/false).

## familyhub.reset_filter
Resets filter counter.

## familyhub.set_power_cool, familyhub.set_power_freeze
Sets Power Cool/Freeze modes (`state`: true/false).
